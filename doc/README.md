# Subtitle workflow document library

이 폴더는 자막 교정·번역 작업에서 다시 사용할 지식을
`AGENT_PROTOCOL.md`와 분리해 보관한다.

## 구조

```text
doc/
├─ index.json              # 에이전트가 먼저 읽는 기준 인덱스
├─ INDEX.md                # 사람이 읽는 인덱스
├─ jobs/                   # 한 영상·한 실행에 대한 검증된 작업 기록
├─ series/                 # 연속 영상·동일 제작자·동일 프로젝트의 연속성 규칙
├─ domains/                # 여러 작업에 재사용할 도메인 용어와 판정 규칙
└─ templates/              # 새 작업 기록 템플릿
```

## 문서 종류

- `job`: 입력, manifest, 참조 문서, 교정 판단, 번역 스타일, 검증 결과와
  발행 위치를 기록한다.
- `series`: 동일 시리즈나 제작자의 말투, 고유명사, 이전 영상과 이어지는
  개념을 기록한다.
- `domain`: Blender 같은 도메인에서 반복되는 용어, 단축키, ASR 오인식
  패턴과 번역 원칙을 기록한다.

## 기준 파일

`index.json`이 문서 ID, 경로, 태그, 연속성 키, 관련 문서와 manifest를
연결하는 기준 인덱스다. `INDEX.md`는 같은 정보를 사람이 빠르게 찾도록
요약한다. 두 파일은 문서를 추가하거나 이동할 때 함께 갱신한다.

과거 문서는 참고 자료일 뿐이다. 현재 영상, 현재 SRT, 현재 manifest와
충돌하면 현재 작업의 직접 증거가 우선한다.
